# IAS Project

## Overview

IAS Project is a modern web application built using Next.js, React, TypeScript, and Tailwind CSS. The application provides a scalable and responsive user interface with reusable components and modern frontend architecture.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Radix UI Components
- Lucide React Icons
- React Hook Form
- Zod Validation
- Recharts
- Next Themes

## Features

- Responsive UI Design
- Reusable Component Architecture
- Theme Support (Light/Dark Mode)
- Form Validation
- Modern Dashboard Components
- Optimized Performance with Next.js
- TypeScript-Based Development
- Scalable Project Structure

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project:

```bash
cd ias_demo
```

Install dependencies:

```bash
npm install
```

## Running the Application

Development Mode:

```bash
npm run dev

Production Build:

```bash
npm run build

Start Production Server:

```bash
npm run start

## Environment Requirements
